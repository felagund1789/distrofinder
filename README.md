# DistroFinder

A small React + TypeScript app (Vite) that lists Linux distributions from `data/distros.json`.

**Quick start**

- Requirements: Node.js >= 20.19 (Vite reports a warning when older).
- Install dependencies:

```bash
npm install
```

- Run the dev server:

```bash
npm run dev
```

- Build for production:

```bash
npm run build
```
